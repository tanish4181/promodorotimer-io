Help & User Guide

Everything you need to know about the Pomodoro Timer & Website Blocker.

The Pomodoro Technique

The Pomodoro Technique is a time management method that uses a timer to break work into intervals, traditionally 25 minutes in length, separated by short breaks. Each interval is known as a Pomodoro. The technique aims to improve focus and reduce mental fatigue.

Timer Controls & Modes

The main popup provides all the controls you need for your sessions.

  • Start/Pause: Begin or pause the current timer.
  • Reset: Reset the current timer to its initial duration.
  • Skip Break: End a break early and immediately start the next focus session.
  • Lock-In Mode (üîí): Commit to a set number of sessions. While active, pause, reset, and skip functions are disabled to enforce focus.
  • Modes: The timer cycles through Focus, Short Break, and Long Break modes automatically. You can see the current mode and session count at a glance.

Focus Tasks (To-Do List)

You can add tasks to a to-do list directly within the popup to keep your goals for the current session front and center. Simply type your task and press Enter or the '+' button. Click the checkbox to mark tasks as complete.

Website Blocking

The extension includes a powerful website blocker to help you stay focused. You can configure it in Settings (‚öôÔ∏è) -> Blocking.

  • Allowlist: Websites on this list are always accessible, even during focus sessions. This is for essential work-related sites.
  • Blocklist: Websites on this list are blocked when the timer is not running (idle). This helps prevent mindless browsing between sessions.
  • Focus Mode Behavior: During a focus session, all websites are blocked *except* for those on your Allowlist.

Break Enforcement

To ensure you take proper mental breaks, the extension can enforce them. When a break starts, a full-screen overlay will appear on all your tabs. This overlay reminds you to step away from the screen and can show a countdown for the remaining break time. You can customize this feature in Settings (‚öôÔ∏è) -> Breaks.

YouTube Integration

To minimize distractions on YouTube, the extension can hide elements like comments, recommendations, and the Shorts feed during your focus sessions. It can also automatically pause any playing video when a break starts. These settings are available in Settings (‚öôÔ∏è) -> YouTube.

  • Data management, including exporting your stats or clearing data.
